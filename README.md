# snake
创意贪吃蛇小游戏

# Play
### 安装字体
> /snake/games/resource/font/HFSnakylines-2.ttf

### 安装依赖
> pip install -r requirements.txt

### 选择人数
- 修改*play.py:14*
> snake_num=1 # 单人

> snake_num=2 # 双人

### 开始游戏
> python play.py

# 其他配置
- width: 宽度
- height: 高度
- bomb_num: 障碍方块个数
- food_num: 食物个数
- unit: 单方块的像素数量
